Admin can lock/unlock or active/deactive user

<Endpoint req="POST">/api/admin/members/:user_id</Endpoint>

#### body:
```js
{
	"active" : 1,
	"locked" : 0
}
```

### Parameters 

#### Optional

| Property | Type | Description |
| ---- | ---- | ---- |
| active | boolean (0,1) | ---- |
| locked | boolean (0,1) | ---- |
