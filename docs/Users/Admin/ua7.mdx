Verify user's document by id.

<Endpoint req="PUT">/api/admin/users/:user_id/verify-documents</Endpoint>


##### body:
```
{
	"verified": 2,
	"documents":[
		{
			"id":95, "status":1
		}
	]
}
```

### Parameters 

## Required
| Property | Type | Description |
| ---- | ---- | ---- |
| verified | ---- | boolean (0,1) |
| documents | ---- | array, its a array of documents 'id' (integer) and 'status' (bolean [0,1]) |
