Update user password by id. Returns true on success. Does not affect the token.

<Endpoint req="PUT">/api/admin/users/password</Endpoint>

##### body:
```js
{
	"id": "3",
	"password": "testing1!"
}
```

### Parameters

## Required
| Property | Type | Description |
| ---- | ---- | ---- |
| id | string | user must exist |
| password | ---- | Password must be minimum eight characters, at least one letter, one number and one special character (testing1!) |
