<Endpoint req="POST">/api/users/authenticate</Endpoint> 

##### body:
```js
{
	"email": "email@gmail.com",
	"password" : "password",
	"for_mobile" : true
}
```
### Parameters 

<Tabs
  defaultValue="require"
  values={[
    { label: 'Require', value: 'require', },
    { label: 'Optional', value: 'optional', },
  ]
}>
<TabItem value="require">

| Property | Type | Description |
| ---- | ---- | ---- |
|`email`|string|email formatted|
|`password`|string|password|

</TabItem>

<TabItem value="optional">

| Property | Type | Description |
| ---- | ---- | ---- |
|`device_type`|string|Platform.IOS(or Android)|
|`device_name`|string|                        |
|`device_id`|string |UDID of the device|
|`device_fcm_token`|string|fcmToken|
|`for_mobile`|boolean| when for_mobile = false the token will expire in 10800 minutes| 

</TabItem>
</Tabs>
