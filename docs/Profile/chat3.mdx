Read all messages of given chatroom

<Endpoint req="POST">/api/users/chatrooms/:chatroom_id/read-all</Endpoint>

## Response
##### 404 Not Found
```
{
    "status": false,
    "message": "ChatRoom not found."
}
```
## Response
##### 200 OK
```
{
    "status": true,
    "message": "All messages are read successfully"
}
```
