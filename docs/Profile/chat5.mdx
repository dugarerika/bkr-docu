Un-Block given user

<Endpoint req="POST">/api/users/:user_id/un-block</Endpoint>

 
 ### Response
 ##### 200 OK
 ```
 {
  "status": true,
  "message": "You have un-blocked given user successfully."
 }
 ```
 
 ##### 422 Unprocessable Entity
 ```
 {
  "status": false,
  "message": "You have not blocked given user"
 }
 ```
 
 ##### 404 Not Found
 ```
 {
  "status": false,
  "message": "User not found."
 }
 ```
