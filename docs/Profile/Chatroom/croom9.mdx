This API will return the user id of avaiable agent with less chat queue.

<Endpoint req="GET">/api/users/chatroom/:chatroom_id/add-members</Endpoint>

#### body:
```js
{
    "participants" : [
        {
			"user_id": 1,
			"participant_id": 2, 
			"participant_type": 1
		},
        {
			"user_id": 2,
			"participant_id": 5, 
			"participant_type": 1
		},
		{
			"user_id": 5,
			"participant_id": 1, 
			"participant_type": 1, 
			"is_admin": true
		}
    ]
}
```
