Add Member to existing chatroom

<Endpoint req="POST">/api/users/chatroom/:chatroom_id/addmembers</Endpoint>

#### body:
```js
{
    "participants" : [
        {
			"user_id": "16f95c92-9b57-4b6b-afc1-1871dc80d926",
			"participant_id": "16f95c92-9b57-4b6b-afc1-1871dc80d926", 
			"participant_type": 1
		}
    ]
}
```

### Parameters 

#### Required
| Property | Type | Description |
| ---- | ---- | ---- |
| participants | array |  |

- e.g  participants : 

```js
        [
            "user_id": Id of user,
            "participant_id": participant id, 
            "participant_type": participant type (1 : User, 2 : Specialist, 3 : Agent, 4 : Business/Place), 
            "is_admin": false
        ]
```
