Create new chatroom with atleast one participant.

<Endpoint req="POST">/api/users/chatrooms</Endpoint>

##### body:
```js
{
    "name": "Hii",
    "description": "",
    "avatar": "",
    "is_group_chat": true,
    "is_chat_with_place": false,
    "is_tech_support": false,
    "solved": false,
    "activity_id": null,
    "participants": [
        {
            "user_id": "05905a94-857d-46fb-a40e-0e8298be29cc",
            "participant_id": "05905a94-857d-46fb-a40e-0e8298be29cc",
            "participant_type": 1,
            "is_admin": false
        },
        {
            "user_id": "9a7ae2aa-944c-4199-9578-38d1a6fb62c3",
            "participant_id": "9a7ae2aa-944c-4199-9578-38d1a6fb62c3",
            "participant_type": 1,
            "is_admin": false
        }
    ]
}
```

### Parameters 

<Tabs
  defaultValue="required"
  values={[
    { label: 'Required', value: 'required', },
    { label: 'Optional', value: 'optional', },
  ]
}>
<TabItem value="required">

| Property | Type | Description |
| ---- | ---- | ---- |
| participants | array |  |

- e.g  participants : 

```js
        [
            "user_id": Id of user,
            "participant_id": participant id, 
            "participant_type": participant type (1 : User, 2 : Specialist, 3 : Agent, 4 : Business/Place), 
            "is_admin": false
        ]
```
</TabItem>
<TabItem value="optional">

| Property | Type | Description |
| ---- | ---- | ---- |
| name | ---- | Chat room Name |
| description | ---- | Description |
| avatar | ---- | Avatar URL |
| is_group_chat | ---- | true or false (default is false) |
| is_chat_with_place | ---- | true or false (default is false) |
| is_tech_support | ---- | true or false (default is false) |
| solved | ---- | true or false (default is false) |
| activity_id | integer | ---- |
| place_id | string | ---- |
| specialist_id | string | ---- |
| contact_user_id | string | ---- |
</TabItem>
</Tabs>
