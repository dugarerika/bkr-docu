Block given user

<Endpoint req="POST">/api/users/:user_id/block</Endpoint>


 
 ### Response
 ##### 200 OK
 ```
 {
  "status": true,
  "message": "You have blocked given user successfully."
 }
 ```
 
 ##### 404 Not Found
 ```
 {
  "status": false,
  "message": "User not found."
 }
 ```
 
 ##### 404 Not Found
 ```
 {
  "status": false,
  "message": "User is blocked already"
 }
 ```
