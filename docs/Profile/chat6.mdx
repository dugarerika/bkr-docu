This API will make user online status to false

<Endpoint req="POST">/api/users/updatepresence</Endpoint>

### Parameters 

## Required

| Property | Type | Description |
| ---- | ---- | ---- |
| is_online | boolean | true or false |

### Response
##### 200 OK

```
{
    "status": true,
    "message": "User presence updated"
}
```

##### 401 Unauthorized
```
{
    "message": "Invalid token",
    "status": false,
    "token": ""
}
```
