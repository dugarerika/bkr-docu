Read given messages

<Endpoint req="POST">/api/users/messages/:message_id/read</Endpoint>

### Response
##### 200 OK

```
{
    "status": true,
    "message": "Message read successfully"
}
```

##### 404 Not Found
```
{
    "status": false,
    "message": "Message not found."
}
```
