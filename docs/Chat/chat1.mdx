Sent message on given chatroom

<Endpoint req="POST">_/api/users/chatroom/:chatroom_id/messages</Endpoint>


 ##### body:
 ```js
 {
  "sender_id" : "600a1a5a-ddd6-40c1-9b79-3ba2178155b0",
  "sender_type": 1,
  "body" : "proof",
  "sent_timestamp" : "020-09-11T13:07:49+00:00"
 }
 ```
 
 ### Parameters 
 
 <Tabs
  defaultValue="required"
  values={[
    { label: 'Required', value: 'required', },
    { label: 'Optional', value: 'optional', },
  ]
}>
<TabItem value="required">

 | Property | Type | Description |
 | ---- | ---- | ---- |
 | body | ---- | message body |
 | sender_id | string | User Id |
 | sender_type | integer | 1 : User, 2 :Specialist, 3: Place, 4 :Support |
 | sent_timestamp | Timestamp | e.g "2020-07-07T07:10:02+0000" |
</TabItem>
<TabItem value="optional">

| Property | Type | Description |
 | ---- | ---- | ---- |
 | has_attachment | boolean | true or false |
 | reference_type | integer | 1 : Webpage, 2: Schedule, 3:Service, 4:Place, 5:Specialist, 6:User, 7: Earning Course |
</TabItem>
</Tabs>
